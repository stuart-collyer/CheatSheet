Semantic HTML Reference

Semantic Element HTML Tag Description
Header <header> Specifies a header for a document or section.
Nav <nav> Defines a set of navigation links.
Main <main> Specifies the main content of the document.
Section <section> Defines a section in a document.
Article <article> Defines independant self contained content.
Aside <aside> This element defines some content aside from the content it is placed in (like a sidebar).
Its content should be indirectly related to the surrounding content.
Figure <figure> Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
Figure Caption <figcaption> Define the caption to go with the Figure element.
Mark <mark> Defines marked/highlighted text
Time <time> Defines a date/time.
Footer <footer> Defines a footer for a document.
